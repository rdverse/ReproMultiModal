The MASUB subroutine is the master subroutine of the package and interfaces
with the user; it performs smooth bivariate interpolation and estimates the
interpolant function at prescribed rectangular grid points in the xâ€”y plane; it
generates a doubly dimensioned array containing these estimated values. The
remaining seven subroutines are supporting and are called by MASUB according
to the values of two appropriate input parameters IC and IEX. EXTRP allows
the user extrapolation outside the convex hull of the point set; it adds some
exterior points to the point set and estimates the function value at those points
with the Shepard method (it is called only by IEX = 1). The choice of this
method has proved to be appropriate for a high percentage of cases; however, if
users find it not sufficiently accurate for their own data, they can set IEX = 0
and add, at least, four other data points (x, y, z) at the corners of the rectangular
domain where the surface has to be reconstructed.
